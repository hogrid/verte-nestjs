# Product Context

## Por que
- Facilitar onboarding do WhatsApp para usuários com feedback claro e confiável.

## Como deve funcionar
- Usuário inicia conexão, lê QR, sistema reconhece automaticamente, mostra sucesso e redireciona.
- Status em tempo real visível no dashboard; botões mudam conforme estado.
- Sincronização de contatos após conexão com feedback de progresso.

## UX Goals
- Mensagens de estado simples: conectando, conectado, desconectado.
- Sem ações ambíguas: esconda “Conectar” quando já conectado; mostre “Desconectar”.
- Streaming de status para UI responsiva.
