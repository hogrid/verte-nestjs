# Verte WhatsApp – Project Brief

## Objetivo
Integrar WhatsApp via Evolution API v2 com suporte a múltiplas instâncias, QR code, status em tempo real, sincronização de contatos e UX clara no frontend.

## Escopo
- Backend NestJS com provider Evolution API.
- Webhook para eventos de conexão e QR.
- Endpoints para status, desconexão e streaming SSE.
- Documentação técnica, versões, e procedimentos de recuperação.
